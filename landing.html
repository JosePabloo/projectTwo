Welcome to The Sandbox!

Start here!

This interface was designed to ease in the process of registration.

Why?

  • Sometimes, when registering for courses, an Augsburg Student has a list of courses he may need but is not sure how they'll fit into a schedule.
  • Some of these classes are higher priority than others...!
  • Some classes may be offered at different times or are held on different days of the week, others aren't.
  • Augsburg's Records & Registration can be difficult to navigate when choosing which classes to take.

The goal: Arrange potential classes to make a desirable schedule without Records & Registration.
Then, export the final schedule of classes to a list that's easy to use as reference when registering.

Here's how it works:

  1. Using Records & Registration, find the classes you're interested in.
    • Be liberal - If there are scheduling conflicts, add them anyway! If there are duplicates, add them all! Everything will be visualized and sorted out in the sandbox.
  2. Add the potential classes to your list of classes in the sandbox.
    • Specify class name, the time and day it's held, and how important it is to take the course this semester.
  3. Visualize your schedule with the drag and drop features.
    • Move things around, see what classes look like in a different arrangement. Find out what classes are held simultaneously.
  4. When you're satisfied, export the schedule to a list of classes.
    • This list will make things MUCH easier when navigating Records & Registration when it is time to sign up for classes!
